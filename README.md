# HubS-TransparentIA

Sistema multi-agente a 5 agenti per la **messa in trasparenza delle competenze** dei volontari, conforme al Decreto Interministeriale 31 luglio 2025.

## Agenti

| # | Nome | Funzione |
|---|------|----------|
| 1 | Orientatore | Raccolta dati, verifica 60h, Patto di Servizio |
| 2 | Stimolatore Riflessivo | Domande Youthpass, autovalutazione EU8/DigComp/LifeComp/EntreComp |
| 3 | Costruttore Dossier | Evidenze, eQval Standard 17, Lever Up 4R |
| 4 | Mappatore Normativo | Competenze → ADA Atlante del Lavoro INAPP |
| 5 | Redattore di Trasparenza | Documento di Trasparenza (Allegato A DI 5/1/2021) |

## Caratteristiche

- ✅ Offline — nessuna API esterna richiesta
- ✅ Sessione JSON — salva e riprendi il lavoro
- ✅ Export DOCX + PDF del Documento di Trasparenza
- ✅ Dati ADA reali da Atlante del Lavoro INAPP (961 ADA)

## Normativa

- DI 31 luglio 2025
- DI 5 gennaio 2021, Allegato A
- GDPR 2016/679
- Raccomandazione UE 2018/C 189/01
